# Binance Futures Testnet Trading Bot (Python)

## Overview
A CLI-based Python application to place Market and Limit orders on Binance Futures Testnet (USDT-M).  
The project demonstrates clean architecture, input validation, logging, and error handling.

## Features
- Market & Limit order support
- BUY and SELL sides
- Binance Futures Testnet (USDT-M)
- CLI input validation
- Structured codebase
- Detailed logging
- Graceful error handling

## Tech Stack
- Python 3.x
- python-binance
- argparse
- logging
